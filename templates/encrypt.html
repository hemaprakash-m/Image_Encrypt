Encrypt Image

This module secures digital images using a hybrid encryption approach. The image pixels are first scrambled using a chaos-based algorithm to disrupt spatial patterns. A symmetric key is then generated and protected using RSA encryption. The encrypted image and key metadata ensure secure storage and controlled access during decryption.
